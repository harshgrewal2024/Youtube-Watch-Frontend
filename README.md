# 🎬 Watch Party App

## 🚀 Live Demo
https://youtubewatchsystem.netlify.app

## 🧠 Features
- Real-time video sync (Play/Pause/Seek)
- Role-based access (Host, Moderator, Participant)
- Kick / Remove users
- YouTube integration
- WebSocket communication

## 🛠 Tech Stack
- Frontend: React + Tailwind
- Backend: Spring Boot + WebSocket
- Deployment: Netlify + Railway
